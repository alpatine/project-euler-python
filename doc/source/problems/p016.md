# Project Euler 16 - Power Digit Sum

## Problem Description
The problem reads:

> $2^{15} = 32768$ and the sum of its digits is $3+2+7+6+8=26$.
> 
> What is the sum of the digits of number $2^{1000}$
>
> [Project Euler Problem 16](https://projecteuler.net/problem=16)

## Approach
Python's support for large integers will be used to solve this problem. The
solution will:
1. Calculate the target number
1. Convert the target number to a string
1. Iterate through each of the digits, for each digit:
    1. convert the digit it an integer
    1. add the digit to a running total

## Python Implementation
``` python
def p16(base: int, power: int) -> int:
    total = 0
    multiple = base ** power
    multiple_str = str(multiple)
    for digit in multiple_str:
        total += int(digit)
    return total

print(p16(2, 1000))
```
